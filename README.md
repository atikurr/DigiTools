# 🛒 DigiTools — Premium Digital Products Store

A modern, fully responsive **digital products marketplace** built with React. Browse, add to cart, and purchase premium digital tools — all in one place.

---

## ✨ Features

- 🛍️ **Add to Cart** — Instantly add products with a single click, with real-time cart count in the navbar
- 🗑️ **Cart Management** — Remove items and view live total before checkout
- 🔔 **Toast Notifications** — Beautiful alerts for every cart action using React Toastify
